# Library Management System (Java)

## Detailed Description

The **Library Management System** is a Java console application designed to simulate the basic operations of a library.

Users can add books, register members, borrow books, return books, and view available books and registered members.

This project demonstrates important **Object-Oriented Programming (OOP)** concepts such as:

* Class design
* Encapsulation
* Lists and collections
* Menu-driven programs
* Modular programming

## Features

*  Add new books
*  Register library members
*  Borrow books
*  Return borrowed books
*  View all books
*  View all members
*  Interactive console menu

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList Collections
* Console Input (Scanner)

## How to Run

Install Java (JDK 8+)

Compile the program

javac Main.java

Run the program

java Main
